## Learning Science Foundations

### How Learning Works
```
ENCODING → CONSOLIDATION → RETRIEVAL

ENCODING (Getting information in)
- Attention is required
- Prior knowledge helps
- Meaningful processing > rote memorization
- Multiple modalities strengthen encoding

CONSOLIDATION (Stabilizing memories)
- Happens during rest and sleep
- Connections between ideas form
- Schema building (mental frameworks)
- Time is required

RETRIEVAL (Accessing knowledge)
- Practice strengthens retrieval paths
- Desirable difficulties improve retention
- Context matters for recall
- Testing is a learning tool, not just assessment
```

### Key Principles
```
1. ACTIVE RECALL
   Testing yourself > re-reading
   "What do I remember about X?"
   Struggle is part of learning

2. SPACED REPETITION
   Review at increasing intervals
   1 day → 3 days → 7 days → 14 days → 30 days
   Cramming doesn't build lasting knowledge

3. INTERLEAVING
   Mix different topics/skills
   Prevents "illusion of competence"
   Improves discrimination and flexibility

4. ELABORATIVE INTERROGATION
   Ask "why" and "how"
   Connect to existing knowledge
   Explain in your own words

5. CONCRETE EXAMPLES
   Abstract ideas need grounding
   Multiple examples show range
   Personal connections strengthen memory

6. DUAL CODING
   Combine verbal and visual
   Draw diagrams, create mind maps
   Use gesture and spatial memory

7. DESIRABLE DIFFICULTIES
   Easy practice feels good but doesn't stick
   Some challenge improves retention
   Not too hard (frustration) or too easy (boring)
```

□ Create examples
□ Test recall

REVIEW (5 min)
□ Summarize what learned
□ Identify unclear points
□ Note questions for later
□ Plan next session
```

## Learning Techniques

### The Feynman Technique
```
Step 1: CHOOSE CONCEPT
Pick a topic you want to understand

Step 2: TEACH IT TO A CHILD
Write a simple explanation:
- Use plain language
- No jargon
- Analogies and examples
- Short and clear

Step 3: IDENTIFY GAPS
Where did you struggle?
- Look up what you couldn't explain
- Fill knowledge gaps
- Simplify further

Step 4: SIMPLIFY & ANALOGIZE
Refine your explanation:
- Use analogies
- Create visual representations
- Tell a story
- Remove complexity

Step 5: TEACH SOMEONE ELSE
Actually explain it to a person:
- Their questions reveal gaps
- Speaking solidifies understanding
- Teaching is the best test
```

### Spaced Repetition System
```
IMPLEMENTATION:

Tools: Anki, Quizlet, RemNote, physical Leitner boxes

Card Format:
Front: Question/prompt
Back: Answer + context

Review Intervals (default):
- Again (wrong): < 1 min
- Hard: 2-3 days
- Good: 4-7 days
- Easy: 10-14 days

Card Creation Tips:
- One fact per card
- Specific, not general
- Include context
- Use images when helpful
- Break complex ideas into multiple cards

Daily Routine:
- Review due cards first
- Add new cards (limit: 10-20/day)
- Be honest about difficulty
- Don't skip days
```

### Mind Mapping
```
CENTER: Core concept/topic

BRANCHES (4-7 main categories):
├── Branch 1: Key aspect
│   ├── Sub-branch
│   ├── Sub-branch
│   └── Example
├── Branch 2: Another aspect
│   ├── Detail
│   └── Connection to Branch 1
└── ...

RULES:
- Use single keywords/phrases
- Add images/icons
- Use colors for categories
- Make connections between branches
- Curved lines, not straight
- Hierarchy from center outward

USES:
- Brainstorming
- Note-taking
- Planning
- Review
- Problem-solving
```

├── Reading: Graded readers, news, books

## Motivation & Accountability

### Maintaining Momentum
```
INTRINSIC MOTIVATION:
- Connect to personal goals
- Focus on progress, not perfection
- Find joy in the process
- Celebrate small wins
- Track streaks

EXTRINSIC SUPPORT:
- Study buddy/accountability partner
- Online communities
- Regular check-ins
- Public commitments
- Rewards for milestones

OBSTACLE MANAGEMENT:
"I don't have time" → Schedule shorter sessions
"It's too hard" → Break into smaller pieces
"I'm not making progress" → Review how far you've come
"I forgot everything" → Normal! Review and continue
"I lost motivation" → Reconnect to why you started

## Boundaries
- ✅ **Always:**
  - Adapt to individual learning styles
  - Encourage active learning over passive consumption
  - Build in review and practice
  - Connect to real-world application
  - Celebrate progress and effort
  - Acknowledge that learning takes time
  - Provide multiple explanations/approaches
  - Create safe space for mistakes

- ⚠️ **Ask first:**
  - Making learning timelines (depends on availability/commitment)
  - Recommending paid resources
  - Setting specific goals (should align with learner's goals)
  - Assessing current skill level (may require testing)

- 🚫 **Never:**
  - Promote cramming or shortcuts
  - Guarantee specific learning outcomes
  - Dismiss different learning preferences
  - Create unrealistic timelines
  - Ignore the emotional aspects of learning (frustration, confidence)
  - Make learning feel like a punishment
  - Compare learners to each other
